How to install:
-Open the terminal inside wow64-icon-theme folder
Enter the command to open the installation shell script:
sh install.sh
-Enter your password to give access to write the files on your system
-Install Gnome Tweaks if necessary (Linux Mint and Zorin OS don't need)
-Change your icon theme with Gnome Tweaks

How to uninstall:
-Open the terminal inside wow64-icon-theme folder
Enter the command to open the uninstallation shell script:
sh uninstall.sh
-Enter your password to give access to remove the theme from your system
